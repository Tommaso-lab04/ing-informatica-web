Feed e dati strutturati

Eco-Nomico espone alcuni endpoint in formato XML: catalogo prodotti, feed RSS, sitemap. Servono per integrazioni con altri sistemi (comparatori di prezzo, aggregatori, motori di ricerca).

Catalogo prodotti

L'intero catalogo in XML. Include un foglio di stile XSLT che il browser applica per mostrarlo come pagina HTML leggibile.

Apri products.xml

Feed RSS

Gli ultimi post della community in formato RSS 2.0. Compatibile con qualsiasi lettore di feed.

Apri rss.xml

Sitemap

Mappa del sito in formato XML standard, utilizzata dai motori di ricerca per l'indicizzazione.

Apri sitemap.xml

I miei ordini (XML)

Esportazione in XML dei tuoi ordini personali. Richiede di essere loggati.

Apri orders.xml

Foglio di stile XSLT

Il foglio di stile XSL che trasforma il catalogo XML in una pagina HTML, aperto come sorgente.
